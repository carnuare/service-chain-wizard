Service Chain Wizard

Import a YAML file to create a service chain in iTop.

Please enter the credentials of the user granted with REST Services User profile.

Example of how the YAML should look like

        
          name: "Service Chain Name"
          description: "Service Chain Description"
          services:
            - name: "Service Name"
              description: "Service Description"
              type: "Service Type"
              subnets:
                - name: "Subnet Name"
                  description: "Subnet Description"
                  cidr: "Subnet CIDR"
                  gateway: "Subnet Gateway"
                  dns: "Subnet DNS"
                  dhcp: "Subnet DHCP"
                  vlan: "Subnet VLAN"
                  type: "Subnet Type"
                  services:
                    - name: "Service Name"
                      description: "Service Description"
                      type: "Service Type"
                      subnets:
                        - name: "Subnet Name"
                          description: "Subnet Description"
                          cidr: "Subnet CIDR"
                          gateway: "Subnet Gateway"
                          dns: "Subnet DNS"
                          dhcp: "Subnet DHCP"
                          vlan: "Subnet VLAN"
                          type: "Subnet Type"
